# Pizza Billing System (Python)

## Description

This is a simple Python-based Pizza Billing System developed for a college lab practical. The program allows the user to select a pizza type, choose add-on items, calculates the total bill, adds GST, applies discounts based on the bill amount, and displays the final payable amount.

## Features

* Select Normal or Deluxe Pizza
* Choose Veg or Non-Veg option
* Add optional items:

  * Extra Cheese
  * Extra Toppings
  * Water Bottles
  * Ketchup Packets
  * Soft Drinks
  * Take Away Charges
* Calculates GST (18%)
* Applies discounts:

  * Above ₹1000 → 5%
  * Above ₹2000 → 10%
  * Above ₹3000 → 12%
  * Above ₹5000 → 20%
* Generates a random bill number
* Displays the final bill

## Requirements

* Python 3.x

## Modules Used

* `random` – To generate a random bill number.

## How to Run

1. Open the program in Python IDLE.
2. Run the program (F5).
3. Enter the required choices when prompted.
4. View the generated bill with GST, discount, and final payable amount.
